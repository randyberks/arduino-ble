# Arduino Bluetooth Low Energy Buzzer Controller
Control a buzzer and led using an android device.
## Compontents used
- Arduino Uno
- ZS-040
- Piezo buzzer
- Red LED
- Breadboard
- 1.5Kohm resistors
- Jumper wires
- BLE app [Bluo](https://play.google.com/store/apps/details?id=com.inspina.bluo&hl=en)
